IOData.hydrogen — whether to read hydrogen atoms

If set to True, then hydrogen atoms are read from PDB, mmCIF or CHARMM files. (By default, only heavy atoms are read.) You will need to turn this on when building all-atom models, although note that the AllHModel class does this for you automatically.
